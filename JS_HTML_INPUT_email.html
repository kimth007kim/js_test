input

input 태그(eMAIL/URL/TEL)

input type=“email” 은 이메일 주소를 입력할 수 있는 입력 필드를 정의합니다.
input type=“url” 은 url 주소를 입력할 수 있는 입력 필드를 정의합니다.
input type=“tel” 은 tel(전화번호)인지를 입력할 수 있는 입력 필드를 정의합니다.

이 속성값은 입력받은 값이 유효한 이메일 주소인지를 자동으로 검증하는 입력 필드를 생성합니다.
이 속성값은 입력받은 값이 유효한 url주소인지를 자동으로 검증하는 입력 필드를 생성합니다.
이 속성값은 입력받은 값이 유효한 tel(전화번호)인지를 자동으로 검증하는 입력 필드를 생성합니다.
